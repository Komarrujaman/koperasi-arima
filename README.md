# Sistem Manajemen Stok Koperasi Kampus

Aplikasi manajemen stok koperasi kampus berbasis web dengan fitur
prediksi kebutuhan stok menggunakan algoritma ARIMA.

## Teknologi
- React.js (Frontend)
- Node.js & Express (Backend)
- MySQL (Database)
- Python (ARIMA)

## Cara Menjalankan

### Backend
```bash
cd backend
npm install
node app.js
